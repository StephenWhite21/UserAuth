# UserAuth

Startup authentication flow for React + Vite, Node + Express, MySQL using JWT

# Setup Instructions

1. Clone the repo

2. Install dependencies:

-   **Frontend**:
    ```
    cd frontend
    npm install
    ```
-   **Backend**:
    ```
    cd backend
    npm install
    ```

3. Create `.env` files in `backend/`

4. Run the server (in `backend/`):
   ```
   npm run dev
    ```

5. Run the server (in `frontend/`):
   ```
   npm run dev
    ```
